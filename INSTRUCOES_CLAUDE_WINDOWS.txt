===============================================================
INSTRUÇÕES PARA CLAUDE CODE - LEIA COM ATENÇÃO
===============================================================

Repositório: https://github.com/limaborges/BOT-NOIA.git

PRIMEIRO: Identificar qual máquina é esta:
- Windows DUAL (CONSERVADORA) = trabalha em conjunto com Linux
- Windows ISOLADA = trabalha sozinha, usa Chrome

===============================================================
SE ESTA MÁQUINA É: WINDOWS DUAL (CONSERVADORA)
===============================================================

PASSO 1: Atualizar código
   git pull

PASSO 2: Criar/editar telegram_config.json com:
   {"enabled": false}

PASSO 3: Criar sync_config.json com:
   {
     "server_ip": "192.168.0.200",
     "server_port": 5555,
     "enabled": true
   }

PASSO 4: Criar machine_config.json com:
   {
     "browser": "firefox",
     "machine_name": "Windows Dual"
   }

PASSO 5: Reiniciar o bot

RESULTADO ESPERADO:
- Telegram NÃO deve iniciar (vai aparecer "Telegram desabilitado")
- Sync com Linux deve funcionar
- Navegador: Firefox

===============================================================
SE ESTA MÁQUINA É: WINDOWS ISOLADA
===============================================================

PASSO 1: Atualizar código
   git pull

PASSO 2: Obter token do @JaceNoiaBot:
   - Telegram → @BotFather → /mybots → JaceNoiaBot → API Token

PASSO 3: Criar/editar telegram_config.json com:
   {
     "token": "TOKEN_DO_JACENOIA_BOT_AQUI",
     "chat_id": "7460639576",
     "enabled": true
   }

PASSO 4: NÃO criar sync_config.json (ou deletar se existir)

PASSO 5: Criar machine_config.json com:
   {
     "browser": "chrome",
     "machine_name": "Windows Isolada"
   }

PASSO 6: Reiniciar o bot

RESULTADO ESPERADO:
- Telegram deve iniciar com @JaceNoiaBot (bot separado)
- Sync deve estar desabilitado
- Navegador: Chrome

===============================================================
ARQUIVOS DE CONFIGURAÇÃO - RESUMO
===============================================================

                    | Windows DUAL    | Windows ISOLADA
--------------------|-----------------|------------------
telegram_config.json| enabled: false  | token do JaceNoiaBot
sync_config.json    | CRIAR (IP Linux)| NÃO CRIAR
machine_config.json | browser: firefox| browser: chrome

===============================================================
PROBLEMAS CONHECIDOS
===============================================================

1. Aposta base pela metade:
   - Zerar reserva_state.json
   - Reiniciar bot

2. Telegram mostra dados errados:
   - Verificar se telegram_config.json está correto
   - Windows Dual: enabled: false
   - Windows Isolada: token do JaceNoiaBot (diferente!)

3. Bot do JaceNoiaBot não responde:
   - Verificar se o token está correto
   - No Telegram: /start no @JaceNoiaBot
